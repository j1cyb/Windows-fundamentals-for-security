# Windows Basics

This section focuses on core Windows tools and components that are commonly used by attackers and frequently investigated by defenders.

The topics in this folder aim to explain how basic Windows functionality can become a security risk when abused, and how Blue Team analysts can identify suspicious behavior.

## What this section covers
- Command-line usage and abuse
- Process and service monitoring
- User and group management
- Common persistence mechanisms
- Lateral movement techniques using built-in Windows features

## Defensive Focus
Each topic is written from a Blue Team perspective and includes:
- Common attacker abuse techniques
- Indicators of suspicious behavior
- Security and detection considerations

## Topics Included
- Command Prompt (CMD)
- Managing Processes
- Windows Registry
- Windows Services
- SMB & RDP
- Scheduled Tasks
- Users and Groups
